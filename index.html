Aiogram History

Aiogram is a Python asynchronous library for developing Telegram bots. It is designed to be easy to use and understand, while also providing advanced features for more complex bot development.

Since its initial release, Aiogram has undergone several updates and improvements, making it one of the most popular choices for Telegram bot development in the Python community.

Why choose aiogram?

Here you can find information about the features, usage, and installation of Aiogram.

  • Asynchronous (asyncio docs, 492)
  • Has type hints (484) and can be used with mypy
  • Supports Pypy
  • Supports Telegram Bot API 7.3 and gets fast updates to the latest versions of the Bot API
  • Telegram Bot API integration code was autogenerated and can be easily re-generated when API gets updated
  • Updates router (Blueprints)
  • Has Finite State Machine
  • Uses powerful magic filters
  • Middlewares (incoming updates and API calls)
  • Provides Replies into Webhook
  • Integrated I18n/L10n support with GNU Gettext (or Fluent)

Installation

From PyPI

pip install -U aiogram

From Arch Linux Repository

pacman -S python-aiogram

Development build (3.x)

From PyPI

pip install -U aiogram

From GitHub

pip install https://github.com/aiogram/aiogram/archive/refs/heads/dev-3.x.zip
